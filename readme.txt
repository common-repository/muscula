Get notified about any WordPress error. Muscula is a logging solution which monitors your website.

== Description ==
Muscula is a logging system which allows you to monitor your app or website written in any language JavaSciprt, TypeScript, C#, Java, Python, PHP and more. You can send errors, logs along with other information and keep it in one place. 

== Installation ==
1. Register at https://www.muscula.com for free
2. Create project and log 
3. Enter LOG ID in plugin configuration.
